# Near API

Rust implementation of Near API library that compatible with `tokio`.

[![Crates.io](https://img.shields.io/crates/v/near-api)](https://crates.io/crates/near-api)

## Usage

First, add this to your `Cargo.toml`:

```toml
[dependencies]
near-api = "0.1.0"
```
